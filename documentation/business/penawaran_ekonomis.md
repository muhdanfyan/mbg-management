# Penawaran Pengembangan Aplikasi MBG Management System

Dokumen ini merangkum biaya pengembangan dan operasional aplikasi MBG Management System.

## Rincian Biaya

| Item | Deskripsi | Biaya (IDR) |
| :--- | :--- | :--- |
| **Biaya Pengembangan** | Jasa pengerjaan aplikasi, integrasi fitur, dan pengembangan sistem (Satu Kali Bayar). | Rp 25.000.000 |
| **Biaya Server & Domain** | Biaya berlangganan server (Cloud/VPS) dan Domain selama 1 tahun. | Rp 2.000.000 |
| **TOTAL** | | **Rp 27.000.000** |

## Ketentuan Pembayaran (3 Termin)

1. **Termin 1 (Uang Muka)**: Sebesar **Rp 7.000.000** dibayarkan di awal untuk mengaktifkan infrastruktur server dan memulai pengerjaan.
2. **Termin 2 (Progress 50%)**: Sebesar **Rp 10.000.000** dibayarkan setelah progres pengembangan mencapai 50% (fitur-fitur utama sudah berjalan di lingkungan terbatas).
3. **Termin 3 (Pelunasan & Serah Terima)**: Sebesar **Rp 10.000.000** dibayarkan setelah seluruh pengembangan selesai, diuji bersama, dan siap dipindahkan ke server produksi.

## Cakupan Pekerjaan
- Pengembangan Backend (Go/Gin) & Frontend (React/TypeScript).
- Integrasi Peta Lokasi (Leaflet) & Sinkronisasi Data SPPG.
- Manajemen Pengguna, Koperasi, dan Rute Distribusi.
- Monitoring Pembangunan & Galeri Foto.

---
*Penawaran ini berlaku sejak tanggal dokumen ini diserahkan. Biaya server tahun berikutnya akan disesuaikan dengan harga pasar vps/server yang digunakan.*
